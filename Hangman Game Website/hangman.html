Hangman Game

Rules:

1. You can guess as many times for the letter but if you get the word wrong three times, the hangman dies and you LOSE :(

2. You will get only one hint: how many letters are in the word. You can only guess one letter at a time or the input box will be cleared! If you repeat a letter you have already guessed, the input box will be cleared.

3. If you guess a letter that is repeated in the word multiple times, you will see the letter repeated that many times in the correct letters box. If you guess a letter that is not in the word, it will be put in the incorrect letters box.

4. You can click on the new word button if you want to play again with a new word or want to try another word. You can know how many incorrect guesses you have used already where it says incorrect guesses.
